# 概要
TOTP認証用コードを生成するためのソフトです。

勉強も兼ねて作ろうといった感じでしたが、バックエンド側は大体AIに書かせました。

## 注意事項
- 安全性は一切保証しません。自己責任でご利用ください。
- 自分が使うために作ったものなので、使いづらいUIになっている可能性が高いです。

## ビルド
### 必須
- Rust
- NodeJS

必要に応じて、[公式ドキュメント](https://v2.tauri.app/start/prerequisites/)を参照して事前準備をしてください。

ビルドするには、以下のコードを実行してください。

```
npm run tauri build
```
